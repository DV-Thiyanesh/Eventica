Решение реальных проблем в цифровую эпоху

Lynked.World — это надежная платформа на основе блокчейна, позволяющая физическим лицам и организациям хранить, контролировать и передавать доверенные цифровые удостоверения, данные и документы.

Lynked.World решает серьезные проблемы, возникающие у людей и организаций в современном цифровом мире:

  • Отсутствие гарантированно безопасного и надежного удостоверения для каждого физического и юридического лица
  • Отсутствие общей платформы для обеспечения надежности и подлинности данных в разных сценариях использования
  • Неэффективность и повышенные расходы, вызванные участием посредников в процессах
  • Невозможность предоставлять доступ к службам различных типов через цифровые платформы по причине уязвимости цифровых удостоверений и связанных с ними данных

Наша экосистема

Экосистема Lynked.World — это комплексное решение, которое предоставляет полноценную платформу приложений, разработанную с нуля, чтобы воспользоваться возможностями технологий блокчейна. Наша платформа основана на блокчейне Ethereum и состоит из многофункциональных модулей, которые решают реальные проблемы, связанные с доверием и подлинностью цифровых удостоверений, документов и данных (как для отдельных лиц, так и для корпораций).

Цифровой кошелек

Цифровой кошелек Lynked.World — удобное средство для хранения удостоверения личности и других документов (например, паспорта, водительских прав, медицинских документов, дипломов и т. д.) на мобильном устройстве. Эти документы можно безопасно верифицировать в блокчейне. С помощью нашего цифрового кошелька пользователи могут хранить и контролировать свои цифровые данные, а также безопасно и удобно передавать удостоверения, документы и данные непосредственно другим лицам, например работодателям, учреждениям и предприятиям.

Основные функции

  • Безопасное хранение цифровых документов на мобильном устройстве.
  • Контроль над данными — передаются только те данные, которые вы явно указали.
  • Проверка документов в блокчейне.
  • Руководители могут отправлять в кошельки пользователей документы с цифровыми подписями.

Сценарии использования

  • Передача удостоверений личности при регистрации в гостиницах, аэропортах и больницах с помощью QR-кодов.
  • Передача работодателям проверенных дипломов и профессиональных сертификатов для устройства на работу.
  • Простая реализация всевозможных услуг, требующих цифровой идентификации.

Передача документов между мобильными устройствами по QR-коду


  1. Создание QR-кода для запросов на получение определенных документов пользователя

    • Запрашивающая сторона сначала выбирает типы документов на настраиваемой странице QR-кода.
    • Запрашивающая сторона нажимает кнопку "Создать QR-код", чтобы создать QR-код, включающий запрос удостоверения и других документов в форме сканируемого QR-кода.

  2. Когда пользователь сканирует сгенерированный QR-код, сведения о запрашивающей стороне и запрашиваемых документах отображаются в мобильном приложении пользователя.

  3. Теперь пользователь может подтвердить запрос и разрешить передачу своих данных и документов запрашивающей стороне.

  4. Авторизация запрашиваемых документов отправляется в цифровой кошелек Lynked.World.

  5. Авторизованные документы вместе с результатами проверки подлинности блокчейна для каждого из них передаются из цифрового кошелька пользователя на запрашивающее устройство.

  6. Запрашивающая сторона получает авторизованные документы, а при необходимости также может проверять подлинность отдельных документов, полученных через блокчейн.

Динамические профили и настраиваемые формы

Предлагаемая нами платформа разработки позволяет любой организации легко создавать формы и развертывать их для использования в нескольких областях. Хороший пример — приложения или формы запросов для различных услуг, которые организации должны предоставлять клиентам через нашу платформу. В это время у пользователей будет доступ к прошедшим проверку подлинности цифровым удостоверениям, созданным в нашей среде благодаря нашей платформе.

Продолжая приведенный выше пример, пользователь может посещать профили этих организаций и оставлять заявки на получение любых доступных услуг, заполняя и подписывая цифровую форму. В качестве дополнительного преимущества платформа также поддерживает автоматическое заполнение форм из профиля пользователя. Любой документ, необходимый для создания заявки, можно автоматически вложить из нашего цифрового кошелька и отправить вместе с подписанной формой.

Настраиваемые формы

Настраиваемые формы Lynked.World через API

  1. С помощью функции "Настраиваемые формы" организации могут создавать формы для различных услуг, предлагаемых ими.

  2. Организации могут либо добавлять эти цифровые формы в свои профили Lynked.World, либо публиковать их на внешних платформах с помощью наших API.

  3. Когда пользователь заказывает какую-либо из этих услуг на нашей платформе, цифровая форма автоматически заполняется данными из цифрового кошелька пользователя. Все необходимые документы также автоматически вкладываются в цифровую форму. При отправке заявки на получение услуги, размещенной на внешней платформе, данные и документы заполняются после того, как пользователь отсканирует QR-код, который включает данные формы и виден на внешней платформе.

  4. После этого пользователь должен добавить цифровую подпись к форме заявки, чтобы завершить ее отправку. Цифровая подпись также хранится в блокчейне.

  5. После отправки заявки необходимые данные и документы отправляются с цифровой подписью пользователя.

  6. Организация, предоставляющая услугу, получает цифровую заявку с запрашиваемыми документами и также может проверить подлинность удостоверения, документов и подписи из формы заявки непосредственно в блокчейне.

Сценарии использования

Реальные сценарии использования

Процесс заселения в гостиницы

  1. Гостиница присоединяется к Lynked.World.

  2. Гостиница генерирует QR-код, включающий требования к удостоверениям и регистрационным документам для заселения.

  3. Постоялец сканирует QR-код с помощью приложения Lynked.World.

  4. Постоялец проверяет и авторизует запрашиваемые документы.

  5. Процесс заселения завершается, когда постоялец отправляет форму со своей цифровой подписью и запрашиваемыми документами.

Заявка на новую услугу (SIM-карту)

  1. Клиент присоединяется к Lynked.World.

  2. Он проходит проверку удостоверения на платформе Lynked.World.

  3. На портале поставщика телекоммуникационных услуг доступна функция регистрации новой SIM-карты с помощью цифрового удостоверения Lynked.World.

  4. При использовании этой функции пользователю предоставляется QR-код, включающий форму заявки на новую SIM-карту со сведениями о необходимом удостоверении и других документах для идентификации клиента.

  5. Клиент сканирует QR-код с помощью приложения Lynked.World.

  6. Клиент проверяет и авторизует запрашиваемые документы и данные для заявки на получение SIM-карты.

  7. Предоставленное удостоверение сверяется с блокчейном через Lynked.World.

  8. SIM-карта выдается и активируется.

Безопасные транзакции

Банки, финансовые учреждения и даже службы розничной торговли в Интернете могут применять функции нашего приложения, чтобы гарантировать подлинность интернет-платежей, операций снятия средств и других транзакций, а также предотвратить использование утерянных или краденых карт и/или удостоверений. Транзакцию в Интернете невозможно завершить, пока пользователь не подтвердит свое цифровое удостоверение с помощью нашего мобильного приложения Lynked.World. Это делает транзакции через Интернет намного безопаснее и надежнее.
