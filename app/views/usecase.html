Check-in Process in Hotels

  1. Hotel joins Lynked.World

  2. Hotel generates QR Code which encapsulates the identity & registration document requirements for Check In

  3. Guest scans the QR code using Lynked.World App

  4. Guest validates and authorizes the requested documents

  5. Check in process is now completed as guest submits the form with his/her digital signature along with the requested documents

 

Application for a new Service (SIM Card)

  1. Consumers joins Lynked.World

  2. Completes his/her identity verification on Lynked.World platform

  3. Telecom Service Provider’s portal has an option to register for a new SIM Card using Lynked.World Digital ID

  4. On using this option, the user is greeted with a QR code that encapsulates the application form for a new SIM Card along with the information about the required identity and other documents for the KYC process

  5. Consumer scans the QR code using Lynked.World App

  6. Consumer validates and authorizes the requested documents and data for the SIM Card application

  7. Provided identity is verified against the blockchain via Lynked.World

  8. Sim Card is Issued and activated

Attendance System

 

A Students Attendance System Using QR Code

Smartphones are becoming more preferred companions to users than desktops or notebooks. Knowing that smartphones are most popular with users at the age around 26, using smartphones to speed up the process of taking attendance by university instructors would save lecturing time and hence enhance the educational process. This paper proposes a system that is based on a QR code, which is being displayed for students during or at the beginning of each lecture. The students will need to scan the code in order to confirm their attendance. The paper explains the high level implementation details of the proposed system. It also discusses how the system verifies student identity to eliminate false registrations.
