解决当前数字时代现实生活中的问题

Lynked.World 是一个基于区块链的安全平台，为个人和机构所有并供其控制和共享数字可信身份、数据及文件。

Lynked.World 解决了当今数字世界中个人与组织面临的关键问题：

  • 每个个体和实体都缺乏有保障的安全、可信的身份
  • 各用例的数据信任与真实性无共同前提条件
  • 因过程涉及中介导致效率低下且成本增加
  • 由于数字身份和相关数据本身的非安全性，无法通过数字平台提供对多种服务的访问

我们的生态系统

Lynked.World 生态系统是一项多方面的解决方案，由完整的应用平台构成，该平台从零开始开发，完善至能够利用区块链技术的功能。我们的平台基于以太坊区块链，包含一系列功能丰富的模块，可围绕消费者和企业等群体的数字身份、文件及数据的信任与真实性解决现实生活问题。

数字钱包

Lynked.World 数字钱包是一种在移动设备上存储个人身份证明及其他文件（如护照、驾驶执照、病历、学位证书等）的便捷方式，。这些文件可通过区块链实现安全验证。我们的数字钱包让用户能够以一种安全便捷的方式拥有和控制自己的数字信息，并直接与雇主、机构和企业等其他实体共享身份、文件和数据。

主要功能

  • 在移动设备上安全存储数字文件
  • 控制自己的数据 — 仅共享您明确授权的数据
  • 通过区块链验证文件
  • 便于当局向用户钱包签发经电子签名的文件

使用案例

  • 在登记入住酒店，进入机场、医院时使用二维码共享身份证件
  • 与雇主共享经验证的教育和职业证书，用于工作申请
  • 轻松使用要求数字识别的各类服务

通过二维码共享文件 移动设备 -> 移动设备


  1. 生成请求用户提供特定文件的二维码

    • 请求者首先从可配置二维码页面中选择文件类型。
    • 请求者点击“生成二维码”代码按钮，生成以可扫描二维码形式融入身份及其他文件请求的二维码。

  2. 当用户扫描生成的二维码时，请求者的详细信息与请求文件将显示在用户的移动应用程序上。

  3. 此时，用户可验证请求并授权与请求者共享他/她自己的数据和文件。

  4. 请求文件的授权现已发送至 Lynked.World 数字钱包

  5. 授权文件以及各自的区块链认证已从用户的数字钱包共享至请求者的设备。

  6. 请求者接收授权文件，同时还可根据需要验证通过区块链接收到的各文件的真实性。

动态概要文件及可配置表单

我们提供的开发平台方便任何组织轻松创建表单，并流畅地部署表单，用于多个领域。用于不同服务（即组织旨在通过我们的平台向消费者提供的服务）的应用程序或请求表单就是一个很好的例子。借助我们的平台，用户可以随时访问在我们的环境中创建的经验证的数字身份。

继续上面的例子，用户可以访问这些组织的概要文件，还可以数字方式填写和签署申请表，从而申请各种可用服务。另一个好处是，该平台还支持使用用户概要文件自动填充的表单。也可自动附上我们的数字钱包中的申请目的所需的任何文件，并与经签署表单一并提交。

基于区块链的

可配置表单

Lynked.World 可配置表单
利用 API 的 Lynked.World 可配置表单

  1. 组织可使用我们的“可配置表单”功能为其提供的各种服务创建表单

  2. 组织可将这些数字表单添加到其在 Lynked.World 上的概要文件中，或使用我们的 API 通过外部平台发布这些表单。

  3. 当用户申请我们平台内提供的任一此类服务时，数字表单将自动填充来自用户数字钱包的数据。所需的任何文件也可自动附加到数字表单中。在申请托管在外部平台上的这类服务时，数据和文件将在用户扫描二维码之后填充，其包含表单的详细信息且在外部平台上可见。

  4. 然后，用户必须对申请表进行电子签名，完成申请流程。电子签名也存储于区块链中。

  5. 申请之后，所需数据及文件将通过用户的电子签名完成提交。

  6. 此时，提供服务的组织可接收包含所请求文件的数字应用程序，并且还可直接通过区块链验证申请表上的身份、文件及签名的真实性。

现实生活用例

酒店登记入住流程

  1. 酒店加入 Lynked.World

  2. 酒店生成二维码，其中包含登记的身份及注册文件要求

  3. 宾客使用 Lynked.World 应用程序扫描二维码

  4. 宾客验证并授权所需文件

  5. 当宾客提交带有他/她的电子签名的表单以及所请求文件时，登记流程就已完成

新服务申请（SIM 卡）

  1. 消费者加入 Lynked.World

  2. 通过 Lynked.World 平台完成他/她的身份验证

  3. 电信运营商的门户网站提供了一个使用 Lynked.World 数字身份注册新 SIM 卡的选项

  4. 使用此选项时，用户会看到一个二维码，其中包含新 SIM 卡的申请表以及有关所需身份的信息和 KYC 流程的其他文件

  5. 消费者使用 Lynked.World 应用程序扫描二维码

  6. 消费者验证并授权 SIM 卡应用程序所需的文件和数据

  7. 通过 Lynked.World 基于区块链验证提供的身份

  8. Sim 卡已发布并激活

安全交易

我们的数字身份解决方案可毫无争议地识别区块链功能使用者的身份，极大地消除欺诈性在线交易风险。

银行、金融机构乃至在线零售服务都可利用我们的应用程序功能，确保在线支付、提款等真实可信，而非使用丢失或被盗的卡和/或身份来完成。在用户使用我们的 Lynked.World 移动应用程序确认他/她的数字身份之前，在线交易将无法完成，由此让互联网交易变得更加安全、更值得信赖。
